# Proyecto de Física

Aplicación web simple desarrollada con HTML, CSS y JavaScript para visualizar contenido/interacciones relacionadas con Física.

## Estructura del proyecto

- `index.html`: estructura principal de la página.
- `styles.css`: estilos visuales.
- `script.js`: lógica e interactividad.

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Cómo ejecutar

1. Descarga o clona este repositorio.
2. Abre el archivo `index.html` en tu navegador.

También puedes usar la extensión **Live Server** en VS Code para verlo con recarga automática.

## Objetivo

Este proyecto sirve como base para prácticas o demostraciones de conceptos de Física en una interfaz web.

## Mejoras futuras (opcional)

- Incluir controles de parámetros (masa, velocidad, aceleración, etc.).
- Mostrar gráficas en tiempo real.
