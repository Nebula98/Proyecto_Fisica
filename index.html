Derivaciones matemáticas formales

Marco teórico de nivel universitario

1) Desde Coulomb puntual hasta campo radial

Partiendo de la fuerza de interacción entre dos cargas puntuales:

F(r) = k * q1 * q2 / r^2

Definiendo campo eléctrico como fuerza por unidad de carga de prueba:

E(r) = F/q0 = k * Q / r^2

La dirección es radial con sentido saliente para Q mayor que 0 e entrante para Q menor que 0.

2) Ley de Gauss para simetría esférica

Con una superficie gaussiana esférica de radio r, E es constante sobre la superficie:

integral(E · dA) = E * 4*pi*r^2 = Qenc/eps0

Despejando:

E(r) = Qenc / (4*pi*eps0*r^2)

La expresión coincide con el resultado de Coulomb al usar k = 1/(4*pi*eps0).

3) Potencial y energía desde el campo

Para una carga puntual en trayectoria radial:

DeltaV = - integral(E · dr) = - integral(kQ/r^2 dr)
V(r) = kQ/r + C

Tomando V(infinito) = 0, C = 0 y la energía potencial queda:

U(r) = qV(r) = kQq/r
